[svelte] 블로그 개설기

주변 개발자가 개발자 블로그를 작성하는 것을 보고 나도 작성할 필요성을 느껴서 만들기로 결심을 했다. 사람들이 블로그를 개설할 때 Tstory나 Velog에서 작성을 하지만 Github Pages를 이용하여 블로그를 개설하는 경우도 있다. Tstory나 Velog는 정해진 틀 안에서 꾸미거나 작성하지만, Github Pages를 이용하면 홈페이지의 디자인부터 UI를 모두 커스텀을 할 수 있다는 매리트를 느껴서 직접 Github Pages를 이용하여 블로그를 만들기로 했다.
